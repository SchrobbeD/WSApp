# Garmin Connect IQ (Forerunner 265) Activity App

This repo is for a Garmin Connect IQ activity app (Monkey C) intended to run on the **Forerunner 265**.

## Local development

- Install the Garmin **Connect IQ SDK Manager**
- Install the **Monkey C** Visual Studio Code extension (Garmin) in Cursor/VS Code
- Build & deploy using the Monkey C extension
